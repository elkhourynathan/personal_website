## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   # or
   pnpm install
   ```
2. **Run the development server:**
   ```bash
   npm run dev
   # or
   pnpm dev
   ```
3. **Open your browser:**
   Visit [http://localhost:3000](http://localhost:3000)
